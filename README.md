# FHIR Subscription Project

Este proyecto implementa un sistema de suscripciones FHIR (Fast Healthcare Interoperability Resources) para la gestión y monitoreo de eventos en el ámbito de la salud.

## 📋 Descripción

El sistema permite la creación, gestión y monitoreo de suscripciones FHIR, facilitando la notificación de eventos relacionados con recursos de salud en tiempo real.

## 🚀 Características Principales

- Gestión de suscripciones FHIR R4
- Soporte para múltiples tipos de recursos
- Sistema de notificaciones en tiempo real
- API RESTful para administración de suscripciones
- Validación de recursos FHIR
- Persistencia de datos
- Logs y monitoreo de eventos
